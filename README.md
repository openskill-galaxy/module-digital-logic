# 数字逻辑电路学习模块 · OpenSkill Galaxy

面向学习《数字电子技术基础》、数字逻辑电路、组合/时序逻辑电路、FPGA 入门、Quartus、BDF 原理图设计与课程设计的学生。

访问地址：<https://openskill-galaxy.github.io/module-digital-logic/>

## 内容范围

数制与编码、逻辑代数、逻辑门、组合逻辑电路（编码器/译码器/数据选择器/加法器/比较器）、触发器、计数器、寄存器、时序逻辑、脉冲波形、存储器、可编程逻辑器件、Quartus、BDF、Verilog、步进电机模拟电梯课程设计。

## 技术栈

Vite + React + TypeScript + Tailwind CSS + React Router + Zustand + Fuse.js + localStorage + 静态 JSON + GitHub Actions。无后端、无数据库、无 AI API。

## 本地开发

```bash
npm install
npm run dev
npm run build
npm run preview
```
